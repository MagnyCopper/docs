开发环境部署

  1. 安装git工具；
  2. 克隆工程到本地磁盘目录git clone https://github.com/MagnyCopper/docs.git;
  3. 安装Node.js最好使用稳定版；
  4. 执行npm安装命令，安装gitbook-clinpm install -g gitbook-cli;
  5. 执行安装gitbook到本地gitbook install；
  6. 运行本地调试命令gitbook serve后，使用浏览器访问http://localhost:4000
